# Neurocommenting Bot

Telegram-бот для автоматического комментирования постов в каналах и группах с помощью ИИ. Управляется через Telegram-интерфейс, поддерживает несколько рабочих аккаунтов одновременно.

---

## Что умеет бот

### Управление ботом
- Список администраторов редактируется прямо из меню бота (раздел **Администраторы бота 👥**) — без правки `config.py`
- Общий прокси меняется из меню (**Общий прокси 🌐**) и сразу применяется к новым подключениям
- Настройка сервисного канала для уведомлений
- Смена токена AI и Agent ID без рестарта
- Добавление дополнительных управляющих ботов

### Управление аккаунтами
- Подключение неограниченного числа Telegram-аккаунтов по номеру телефона
- Главное меню каждого аккаунта показывает статусы всех функций: сколько каналов подключено, включены ли автоответы, рассылка и аутрич
- Редактирование профиля аккаунта: имя, фамилия, фото, описание, юзернейм
- Настройка индивидуального SOCKS5 прокси для каждого аккаунта
- Удаление аккаунта

### Нейрокомментинг ⚙️
- Автоматически комментирует новые посты в указанных каналах и группах
- Генерирует уникальные комментарии через ИИ (ChatGPT-совместимый API)
- Режим **"Комментировать первым"**: занимает место placeholder-сообщением, затем редактирует его готовым текстом
- Режим **"Шаблонные комментарии"**: вместо ИИ выбирает случайный текст из заданного списка
- Настройки (в подменю **Нейрокомментинг ⚙️** каждого аккаунта):
  - **Роль** — системная роль для ИИ (например: "Вы — маркетолог")
  - **Промпт** — задание для ИИ (например: "Напиши короткий комментарий на тему этого поста")
  - **Задержка** — пауза в секундах перед комментарием после выхода поста
  - **Шанс** — вероятность оставить комментарий (1 к N)
  - **Мин. символов** — минимальная длина поста; более короткие посты пропускаются
  - **Каналы** — добавить/удалить каналы для мониторинга (бот автоматически вступает при добавлении)

### Автоответы 💬
- Автоматически отвечает на входящие личные сообщения воркеру
- Включаются и отключаются кнопкой в шапке раздела **Автоответы 💬**
- Бот-аккаунты (спам-боты) автоматически игнорируются
- Два режима:
  - **Статичный текст** — всегда отвечает одним и тем же заранее заданным сообщением
  - **ИИ-диалог** — ведёт полноценный диалог: отдельная роль и промпт, история переписки с каждым пользователем, настраиваемый лимит раундов

### Аутрич в ЛС 📨
- Воркер отслеживает указанные группы и автоматически пишет в личку всем, кто там пишет
- Никогда не пишет одному человеку дважды (дедупликация по всей истории)
- Настройки: текст сообщения, лимит DM в день, задержка перед отправкой, список групп для мониторинга
- При достижении дневного лимита останавливается до следующего дня

### Инвайтинг 🦁
- Массовое добавление пользователей в группу с настраиваемым интервалом между приглашениями

### Рассылка по ЛС 🌪
- Массовая отправка сообщений по заданному списку пользователей

### Рассылка в чаты 📢
- Воркер рассылает заданный текст в список чатов в течение дня со случайными интервалами
- Работает только с **10:00 до 18:00 МСК** — вне окна засыпает до следующего утра
- Интервал задаётся в формате `мин-макс` секунд (например `600-1500` — от 10 до 25 минут)
- Включается и отключается кнопкой в шапке раздела **Рассылка в чаты 📢**
- После каждой успешной отправки приходит уведомление в сервисный канал

## Требования

- VPS с Ubuntu 22.04 (минимум 1 CPU, 1 GB RAM)
- Python 3.10+
- Аккаунт на [my.telegram.org](https://my.telegram.org) для получения API_ID и API_HASH
- Telegram-бот созданный через [@BotFather](https://t.me/BotFather)
- Аккаунт у AI-провайдера с ChatGPT-совместимым API

## Развёртывание на VDS

### 1. Создай VPS

На любом VPS-хостинге → создай сервер:
- ОС: Ubuntu 22.04
- Тариф: минимальный (1 CPU, 1 GB RAM хватает для 5–10 аккаунтов)
- Запомни IP и пароль root

### 2. Подключись по SSH

```bash
ssh root@<IP_СЕРВЕРА>
```

### 3. Установи зависимости

```bash
apt update && apt install python3 python3-pip python3-venv git -y
```

### 4. Склонируй репозиторий

```bash
git clone https://github.com/tw2igor/neurocommenting-bot /root/ComBot
cd /root/ComBot
```

### 5. Создай виртуальное окружение и установи библиотеки

```bash
python3 -m venv venv
source venv/bin/activate
pip install requests xlsxwriter tgcrypto
pip install "git+https://github.com/KurimuzonAkuma/pyrogram/archive/dev.zip"
```

> Используется форк Pyrogram от KurimuzonAkuma — он поддерживает актуальные функции Telegram API.
> `tgcrypto` — опциональная C-библиотека для ускорения MTProto-шифрования. Без неё бот работает, но медленнее.

### 6. Получи Telegram API credentials

Зайди на [my.telegram.org](https://my.telegram.org) → API development tools → создай приложение. Запомни `api_id` и `api_hash`.

### 7. Создай управляющего бота

Напиши [@BotFather](https://t.me/BotFather) → `/newbot` → задай имя и username. Получи токен вида `123456789:AAxxxxx...`.

Чтобы узнать `BOT_ID` (числовой ID бота) — возьми числовую часть токена до двоеточия: `123456789`.

### 8. Получи AI credentials (например в Timeweb)

Зарегистрируйся у AI-провайдера с ChatGPT-совместимым API:
- Создай AI-агента и скопируй **Agent Access ID** (`AI_AGENT_ID`)
- В настройках профиля создай **Bearer-токен** (`AI_TOKEN`)

### 9. Настрой конфигурацию

Скопируй пример конфига и заполни своими данными:

```bash
cp config.example.py config.py
nano config.py
```

Содержимое `config.py`:

```python
# Telegram App (my.telegram.org)
API_ID = 12345678
API_HASH = 'your_api_hash_here'

# Управляющий бот (@BotFather)
BOT_ID = 123456789
BOT_LINK = 'your_bot_username'        # без @
BOT_TOKEN = '123456789:AAxxxxxxxx...'

# Telegram ID администраторов бота
ADMINS = (123456789,)                 # твой Telegram ID, можно несколько через запятую

# AI API
AI_TOKEN = 'eyJ...'
AI_AGENT_ID = 'xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx'
```

> `config.py` не попадает в git — он в `.gitignore`.

Узнать свой Telegram ID можно через [@userinfobot](https://t.me/userinfobot).

### 10. Настрой автозапуск через systemd

```bash
cp systemd/com.service /etc/systemd/system/
systemctl daemon-reload
systemctl enable com
systemctl start com
```

### 11. Проверь запуск

```bash
systemctl status com
```

Логи в реальном времени:

```bash
journalctl -u com -f
```

## Первый запуск

1. Напиши своему боту `/start` в Telegram — должно появиться главное меню
2. Нажми **"Добавить аккаунт"** и следуй инструкции:
   - Введи номер телефона в международном формате (`+79001234567`)
   - Введи код подтверждения из Telegram
   - Если включена двухфакторная аутентификация — введи пароль
3. После добавления аккаунт появится в разделе **"Мои аккаунты"**
4. Зайди в настройки аккаунта → **Нейрокомментинг ⚙️** и настрой:
   - **Роль** — кем представляется ИИ
   - **Промпт** — что именно писать в комментариях
   - **Задержку** и **шанс** по желанию
5. Там же нажми **"Добавить каналы 💬"** и отправь ссылки на каналы/группы, каждую с новой строки
6. Бот начнёт автоматически комментировать новые посты

## Частые вопросы

**Бот запустился, но не комментирует**
- Убедись что добавил каналы через "Добавить каналы 💬" в настройках аккаунта
- Проверь что аккаунт вступил в канал (бот делает это автоматически при добавлении)
- Посмотри логи: `journalctl -u com -f`

**Ошибка при добавлении аккаунта**
- Номер должен быть в международном формате: `+79001234567`

**Как добавить или удалить администратора**
- В меню бота → **Меню 🌴** → **Администраторы бота 👥** — список с кнопками удаления и кнопкой «Добавить ➕»
- Изменения применяются сразу, без рестарта
- При первом запуске список инициализируется из `config.py` (поле `ADMINS`)

**Как настроить прокси для аккаунта**
- В настройках аккаунта есть кнопка прокси
- Формат: `socks5:host:port` или `socks5:host:port:user:pass` для аутентифицированного

**Как настроить глобальный прокси (для всех подключений)**
- В `config.py` задай поле `PROXY`:
  ```python
  PROXY = 'socks5:45.145.57.238:10253'             # анонимный
  PROXY = 'socks5:45.145.57.238:10253:user:pass'   # с авторизацией
  ```
- Глобальный прокси применяется ко всем клиентам — менеджер-боту, воркерам, добавлению аккаунтов
- Если у аккаунта задан свой прокси — он имеет приоритет над глобальным
